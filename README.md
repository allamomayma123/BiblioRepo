# Gestion d'une bibliothèque

## Description
Cette application permet de gérer une bibliothèque en utilisant MongoDB et Neo4J pour le stockage des données, ainsi qu'un module d'authentification pour sécuriser l'accès.

## Installation
1. Cloner le repository.
2. Installer les dépendances:
    ```bash
    pip install -r requirements.txt
    ```
3. Configurer les paramètres dans `config.py`.
4. Lancer l'application:
    ```bash
    python run.py
    ```

## Utilisation
Accéder à l'application via `http://localhost:5000`.
